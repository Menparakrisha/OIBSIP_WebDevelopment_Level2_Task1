# OIBSIP_WebDevelopment_Level2_Task1
Calculator Web App using HTML, CSS &amp; JavaScript | OIBSIP Web Development Internship Level 2 Task 1

Modern Calculator Web Application
📌 Project Overview

This project is a Modern Calculator Web Application developed using HTML, CSS, and JavaScript as part of the Oasis Infobyte (OIBSIP) Web Development Internship – Level 2 Task 1.

The calculator provides a clean and responsive user interface for performing basic arithmetic operations such as addition, subtraction, multiplication, division, percentage calculations, and decimal operations. The project demonstrates core front-end development concepts including DOM manipulation, event handling, and responsive design.
